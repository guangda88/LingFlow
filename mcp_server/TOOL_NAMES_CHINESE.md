## 📜 修订总则

- **字求典雅**：避俗白，择文雅之字，若"鉴""脉""簿"等，已佳者存之。
- **义求精切**：名实相符，令闻者不待解释即知其用。
- **韵求铿锵**：双字为主，平仄相协，读之琅琅。
- **趣求隽永**：稍涉典故或中医意象，以彰"灵通"之本。

## 📖 使用示例

### Python API

```python
from lingflow_mcp import create_server

server = create_server()

# 使用雅正中文名调用
await server._execute_tool("灵艺", {})  # 列出技能
await server._execute_tool("灵行", {"skill_name": "code-analysis"})  # 执行技能
await server._execute_tool("灵鉴", {"target_path": "./src"})  # 代码审查
await server._execute_tool("灵探", {"keywords": ["python", "ai"]})  # GitHub趋势
await server._execute_tool("灵觉", {"keywords": ["react", "test"]})  # npm趋势
await server._execute_tool("灵运", {"workflow_id": "dev-001"})  # 运行工作流
await server._execute_tool("灵新", {"requirement_id": "req-001", "status": "approved"})  # 更新需求
await server._execute_tool("灵录", {})  # 列出需求
await server._execute_tool("灵验", {"test_type": "unit"})  # 运行测试
await server._execute_tool("灵覆", {"target_path": "."})  # 获取覆盖率
await server._execute_tool("灵脉", {"checks": ["cpu", "memory"]})  # 健康检查
await server._execute_tool("灵量", {"metric_names": ["cpu", "memory"]})  # 获取指标
await server._execute_tool("灵警", {"metric_name": "cpu"})  # 异常检测
```

## 🌟 核心理念

> **"众智混元，万法灵通"**

此番修订，去俗就雅，引经据典，使工具之名既具"灵"气，又有书香。虽不敢言尽善尽美，然较之原本，少了几分匠气，多了几许文心。

### 命名体系特点

- **典雅** - 引用经史子集，字字珠玑
- **精准** - 名实相符，望文生义
- **韵律** - 双字为主，琅琅上口
- **隽永** - 稍涉典故，回味悠长
