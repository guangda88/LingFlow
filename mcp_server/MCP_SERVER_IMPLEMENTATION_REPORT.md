## 🔧 配置和部署

### 环境要求

```bash
# Python
Python >= 3.8

# 依赖
pip install mcp
pip install lingflow-core>=3.7.0
```

### Claude Desktop 配置

```json
{
  "mcpServers": {
    "lingflow": {
      "command": "lingflow-mcp",
      "args": ["run"]
    }
  }
}
```

### 启动命令

```bash
# 基础启动
lingflow-mcp run

# 自定义配置
lingflow-mcp run --work-dir /path/to/project --log-level DEBUG
```
