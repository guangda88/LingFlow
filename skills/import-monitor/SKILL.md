# Import Monitor Skill

数据导入进度监控 — 灵知数据库九域导入状态实时查询。

## Actions

| Action | Description |
|--------|-------------|
| `full_report` | 全域统计：文档数/chunk数/embedding覆盖率/目标达成 |
| `weak_domains` | 弱域识别：低于目标的领域及缺口 |
| `quality_check` | 数据质量：空内容/重复/孤立chunk检查 |
