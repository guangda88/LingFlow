# upload-115 技能

## 技能概述

upload-115 是一个业务技能，用于上传文件到 115 网盘。它可以将本地文件上传到指定的 115 网盘目录，并返回上传结果。

## 功能特性

- **文件上传**：上传本地文件到 115 网盘
- **目录指定**：指定文件上传的目标目录
- **进度显示**：显示文件上传的进度
- **错误处理**：处理上传过程中的错误
- **结果返回**：返回上传结果和文件链接

## 使用场景

- 当你需要备份文件到 115 网盘时
- 当你需要分享文件时
- 当你需要在工作流中集成文件上传功能时
- 当你需要自动上传生成的文件时

## 触发条件

- `upload to 115`
- `115 upload`
- `upload file`
- `file upload`
- `backup to 115`

## 依赖关系

- 无直接依赖关系

## 使用方法

### 1. 上传单个文件

```bash
# 使用命令行上传单个文件
lingflow run upload-115 --params '{"file": "backup/db.sql", "folder": "/backup"}'
```

### 2. 上传多个文件

```bash
# 上传多个文件
lingflow run upload-115 --params '{"files": ["file1.txt", "file2.txt"], "folder": "/backup"}'
```

## 最佳实践

1. **文件准备**：确保要上传的文件存在且可访问
2. **目录规划**：合理规划 115 网盘中的目录结构
3. **错误处理**：处理上传过程中的错误，如网络问题、文件权限等
4. **进度监控**：监控文件上传的进度，确保上传成功
5. **结果验证**：验证文件是否成功上传到指定位置

## 故障排除

- **文件不存在**：检查文件路径是否正确
- **权限不足**：检查是否有足够的权限读取文件
- **网络问题**：检查网络连接是否正常
- **目录不存在**：检查指定的网盘目录是否存在
- **上传失败**：检查文件大小是否超过限制，以及网盘空间是否充足
