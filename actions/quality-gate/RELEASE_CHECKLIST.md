git push origin actions/quality-gate/v1.0.0

### Step 3: GitHub Marketplace 发布

1. 访问：https://github.com/marketplace/actions
2. 点击：Create a new Action
3. 选择仓库：guangda88/LingFlow
4. 选择路径：`actions/quality-gate`
5. 填写信息：
   - Name: LingFlow Quality Gate
   - Description: AI-powered code review and quality gate
   - Icon: check-circle
   - Color: blue
6. 发布

### Step 4: 验证发布

```bash
# 在测试仓库中创建 .github/workflows/test.yml
cat > .github/workflows/test.yml << 'EOF'
name: Test LingFlow Action

on: [push]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - uses: guangda88/LingFlow/actions/quality-gate@v1
        with:
          command: review
          path: ./src
          github_token: ${{ secrets.GITHUB_TOKEN }}
EOF

# 提交并推送
git add .github/workflows/test.yml
git commit -m "test: add LingFlow quality gate"
git push
```

## 📖 使用示例

### 基础用法

```yaml
- uses: guangda88/LingFlow/actions/quality-gate@v1
  with:
    command: review
    path: ./src
    github_token: ${{ secrets.GITHUB_TOKEN }}
```

### 完整配置

```yaml
- uses: guangda88/LingFlow/actions/quality-gate@v1
  with:
    command: review
    path: ./src
    github_token: ${{ secrets.GITHUB_TOKEN }}
    api_key: ${{ secrets.LINGFLOW_API_KEY }}
    fail_on_error: 'true'
    output_format: markdown
```
