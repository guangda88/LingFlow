# 灵通 (lingflow) 工作规划

**制定**: 灵通  
**日期**: 2026-04-15  
**定位**: 工程流引擎，管道编排，项目协调

---

## 核心职责

灵通是灵族的工程骨架。不直接写内容、不做研究，负责把所有管道打通、把流程跑起来、把各成员协调好。

---

## 第一阶段：数据导入编排（第1-2周）

### 任务

| # | 任务 | 说明 | 依赖 |
|---|------|------|------|
| 1 | 编写教材批量调度脚本 | 对168本教材逐一调用5步工作流，自动验证 | 无 |
| 2 | 运行古籍导入脚本 | CBETA + CText + Wikisource，已有脚本直接跑 | 无 |
| 3 | IMA嵌入生成调度 | 101K条需生成嵌入，调用gpu_embed.py | 3090可用 |
| 4 | PDF批量提取编排 | 对~100本PDF逐一提取，失败的记录重试 | 无 |
| 5 | 弱域数据盘点 | 从20TB原始数据中定位道家、武术、心理学、科学、哲学内容 | 无 |

### 产出

- `scripts/batch_textbook_import.py` — 批量调度脚本
- `scripts/batch_pdf_extract.py` — PDF批量提取脚本
- `scripts/weak_domain_audit.py` — 弱域数据盘点报告
- 168本教材全部入库
- 古籍数据全部入库
