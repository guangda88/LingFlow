# 个人简介

刘庆，男，原解放军八十八医院中医科主任，主任医师，中医学博士后，二级心理咨询师。毕业于第二军医大学。国家中医药管理局中医大师传承人才培养计划学员，师从颜德馨、邓铁涛、路志正、朱良春、周仲瑛、张琪等六位国医大师。担任中国中西医结合诊断专业委员会常务委员、全军心身医学专业委员会常务委员等职。出版专著2部，发表论文30余篇，主研国家级课题1项，获得省部级科技二等奖1项。自2020年初跟随管成学教授进行《宋史》人物苏颂的研究工作，主持校注《苏颂全集·苏魏公文集》（福建教育出版社）。目前参与校订庞鹤鸣教授系列出版物，并组织整理相关历史资料。

AI架构师，lingflow（灵通）智能工作流引擎创始人。致力于将生命整体论及道家哲学思维应用于人工智能架构与安全研究，探索"自觉·自决·进化"的系统设计理念。研究课题包括基于技能驱动的多Agent协调架构、AI系统的元认知与知识边界管理、以及自我验证机制的工程化实现。独立负责灵家（灵通、灵依、智桥）全系产品的架构设计与运维部署。
