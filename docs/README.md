## 文档维护说明

### 新增文档
1. 在本目录中创建新的 Markdown 文档
2. 使用 `snake_case_with_underscores.md` 命名格式
3. 更新本 README 的对应分类部分
4. 确保文档标题包含版本信息和最后更新日期

### 文档格式
- 使用标准 Markdown 语法
- 标题层级：`#` → `##` → `###` → `####`
- 代码块使用语言标识：```python```、```bash``` 等
- 表格使用标准 Markdown 表格语法
- 链接使用相对路径：`./filename.md`

### 文档更新
- 每次更新文档时，更新文档头部的日期和版本信息
- 在文档末尾添加更新日志
- 重大更新时通知相关团队成员
