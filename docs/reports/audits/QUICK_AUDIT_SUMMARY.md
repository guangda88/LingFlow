## 💡 使用方法

### 自动模式（已启用）

```python
from lingflow.bootstrap import bootstrap

# 启动lingflow（hooks自动启用）
status = bootstrap(hooks=True)

# hooks会自动检测并提示优化
# 无需手动干预
```

### 手动检查

```bash
# 检查是否需要优化
lingflow optimize check

# 运行优化
lingflow optimize structure --target ./

# 查看报告
cat LINGFLOW_OPTIMIZATION_*.md
```
